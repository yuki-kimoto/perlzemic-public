INT16_MAX - int16_tの最大値を表す定数

INT16_MAXはint16_t - 符号あり16bit整数型の最大値を表す定数です。「stdint.h」ヘッダをインクルードすることで使えます。C99で導入された定数マクロです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%d", INT16_MAX);
}

printf関数のフォーマット指定子で「%d」を使っています。

出力されたINT16_MAXの値です。 

32767
