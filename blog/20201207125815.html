コメント

C言語のコメントは「/*」で始まり「*/」で終わります。コメントは、ソースコードとしては意味を持ちません。 

/* コメント */

複数行コメントも書けます。 

/*
  コメント1
  コメント2
*/

C99で採用された一行コメント //

一行コメント「//」がC99で採用されました。 

// コメント
