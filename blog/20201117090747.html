int8_t - 符号あり8bit整数型

「int8_t」は符号あり8bit整数型です。「stdint.h」ヘッダをインクルードすることで使えます。C99で導入された型です。 

# 符号あり8bit整数型
int8_t

符号あり8bit整数型が表現できる整数の最大値は「127」、最小値は「-128」です。

最大値は「INT8_MAX」、最小値は「INT8_MIN」というマクロで定義されています。

int8_tのサンプルコード

int8_tを使ったサンプルコードです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  int8_t num = 5;
  
  printf("%d\n", num);
}

printf関数のフォーマット指定子で「%d」を使っています。

出力結果。 

5
