UINT8_MAX - uint8_tの最大値を表す定数

UINT8_MAXはuint8_t - 符号あり8bit整数型の最大値を表す定数です。「stdint.h」をインクルードすると利用できます。「stdint.h」ヘッダをインクルードすることで使えます。C99で導入された定数マクロです。 

#include <stdio.h>
#include <stdint.h>

int main(void) {
  printf("%u", UINT8_MAX);
}

printf関数のフォーマット指定子で「%u」を使っています。

出力されたUINT8_MAXの値です。 

255
